Yucibot features

Point system

Viewers can earn points by watching the stream. By default you earn 5 points every 5 minutes. With these points users can buy commands and gamble to earn even more. You can do this by using the roulette & slot command. Another way to earn points is to participate in raffles, which are started with every new subscriber or can be started by mods. By earning points you can also climb the leaderboards and show off your points to others!

Stat tracking

Everything you do in chat is recorded by the bot. For example, it keeps track of your lines, points and and a whole lot more. One of the special features is that the bot can track the emote usage of the chat. You can view these stats on the stats page

Moderation tools

Twitch chatbots always come with some handy mod tools. The bot currently is able to detect links in chat. You can also filter certain words from the chat. The filter will find any word in a messages and punishes the user by a purge, time-out or ban.

Songrequest

Users are able to request songs in chat. By default only subs are able to request songs, but this can be changed if the streamer wants to. You can also change the max length of a song or the max amount of songs in the queue. If a song exceeds the max length, you are still able to request is, but it will get skipped at the max timestamp.

Custom commands

You can add your own commands in the bet. You are able to adjust the cooldown, cost and level of the command, so you can have sub only commands as example.

Chat logger

Every message that someones types in the chat will be logged and stored into the database. You can retrieve a users log by going to their profile page on the website. Selecting a certain date will retrieve the logs for that day. You are also able to retrieve the entire user log, but this is not recommended.
